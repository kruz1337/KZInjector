### Injection Types
- LoadLibrary
- ManualMapping
### Execution Methods
- CreateRemoteThreadEx
- ThreadHijacking

## How to enable Auto Injection?
* First open settings.xml file with notepad.

* Then set "enabled" next to "AutoInject" to "true".

* Configurate "AutoInject" settings.

* That's all, enjoy it :)
